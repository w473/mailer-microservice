# mailing-microservice
## Description
Microservice for sendings emails


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run tests

# e2e tests
$ npm run tests:e2e

# tests coverage
$ npm run tests:cov
```
